# PrettierPrint

A drop-in replacement for the `prettyprint` gem with more functionality.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "prettier_print"
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install prettier_print
